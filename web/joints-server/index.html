ねこみんMCServer

ねこみんMCServerの参加

アドレスはこちらから

  • サーバーアドレス:minecraft.nekomin.jp (java/統合)
  • ポート番号:初期値

統合版での参加はmojangアカウント・Microsoftアカウント
のどちらかが必要です


詳しい参加方法は下へ読み続けてくださいで読んでください

サーバーへの参加方法

JAVAでの参加の仕方

参加の仕方は

  1. マルチプレイを選択
  2. サーバーを追加を選択
  3. サーバーアドレスにminecraft.nekomin.jpと入力
    サーバー名はおまかせ

統合版(パソコン/スマホ)の参加方法

  1. 遊ぶを選択
  2. サーバーを選択
  3. 下へスクロールしてサーバーを追加を選択
  4. サーバーアドレスにminecraft.nekomin.jpと入力
    サーバー名はおまかせポートは変更しないでください

Switch™/PlayStation®4&5
統合版の参加方法

  1. インターネット設定でDNS設定を
    優先DNSを104.238.130.180に
    代替DNSを008.008.008.008に変更してください
  2. Minecraftを起動し遊ぶを選択してサーバーを選択
  3. 特集サーバーのどちらかを選択してください
  4. 入ったら一番上の「Connect to a Server」を選択してください
  5. 一番上の「Server Address」にminecraft.nekomin.jpと入力し真ん中は変更しないでくださいあとは、一番下はおまかせで入力しスイッチを右にしてボタンを押してくださいあとは勝手に入ります。
